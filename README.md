# Shubham Rawat Portfolio Website

A responsive personal portfolio website built with HTML, CSS, and JavaScript. It presents my profile as a Full Stack Developer with experience in .NET, SQL Server, JavaScript, jQuery, Bootstrap, .NET MAUI, and enterprise workflow modules.

## Live Sections

- Hero introduction with resume link
- About section based on my full stack development profile
- Technical skills grouped by frontend, backend, database, and tools
- Professional experience at Cyborg ERP Pvt Ltd
- Project/use case cards from real development work
- Education details
- Contact links for email, phone, LinkedIn, and GitHub

## Project Use Cases

### Leave Apply Form

A digital leave request module that allows students to submit and track leave requests. The use case focuses on replacing manual approval workflows with a structured online form and status-driven process.

### Admission Letter Automation

A webpage that generates and sends admission confirmation letters through email after a student's admission is approved. This use case supports faster communication and reduces repetitive manual document handling.

### Parking Pass System

A webform for assigning and generating vehicle parking passes for students and staff. The use case includes organized record handling for vehicle and user details.

### Dress Distribution Module

A uniform/dress issuance module for assigning items to students, tracking distribution, maintaining inventory records, and generating reports to prevent duplication or errors.

### Cross-Platform Mobile App

A .NET MAUI mobile application with SignalR-based real-time notifications. The use case focuses on instant user updates and better engagement across mobile platforms.

### Payment Module Integration

Payment modules for student enquiry registration and event registration forms. The use case includes payment gateway flow updates, new gateway integration, and redirecting students to the selected payment gateway.
